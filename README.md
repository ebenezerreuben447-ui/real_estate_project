# Real Estate Price Prediction App 🏠

## Overview
This project is a **Real Estate Price Prediction app** built with Python. It combines **Machine Learning**, **FastAPI**, and **Streamlit** to provide users with a simple interface for predicting house prices based on key features.

The app showcases:
- Data analysis and cleaning
- Model training and evaluation
- FastAPI backend for predictions
- Streamlit frontend for user interaction

---

## Features
- Explore and analyze real estate data
- Predict house prices using a trained ML model
- Easy-to-use web interface via Streamlit
- Secure database connections using `.env` for credentials

---

## Technologies Used
- **Python** – core programming language
- **Pandas & NumPy** – data manipulation
- **Scikit-learn** – machine learning
- **FastAPI** – backend API for predictions
- **Streamlit** – frontend UI
- **MySQL** – database (connection hidden with `.env`)
